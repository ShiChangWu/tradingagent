## Download TradingAgents Advisor 2.0

### Windows Portable (v2.0.0)

**File:** `TradingAgentsAdvisor_Setup.zip` (~191 MB)

### Installation

1. Download the zip file
2. Extract all files to a folder
3. Run `install.bat` (double-click)
4. The installer will copy files to `%LOCALAPPDATA%\TradingAgentsAdvisor` and create Desktop + Start Menu shortcuts
5. Launch from the shortcut or run `TradingAgentsAdvisor.exe` directly

### Manual Setup

If you prefer manual setup:

1. Extract the zip to any folder
2. Copy `.env.example` to `.env` and configure your settings
3. Run `TradingAgentsAdvisor.exe`

### Requirements

- Windows 10 or later (64-bit)
- No additional dependencies required (self-contained)
